Purpose

The X-CUBE-AWS (STM32 AWS IoT software expansion for STM32Cube) provides a port of the Amazon FreeRTOS operating system to STM32WB55.Nucleo board.

It uses Amazon FreeRTOS support for BLE and relies on a mobile phone application as a proxy to AWS services (via mobile network or WiFi).

Both the aws_demos and the aws_tests reference applications of FreeRTOS are provided.

Known Limitations

  • The compilation warnings of the bootloader applications are due to the disabled security features.

  • The compilation warnings of the user applications are due to the FreeRTOS code or to to its dependencies

  • Programming the STM32WB55.Nucleo board with STM32CubeIDE does not work. After compiling with CubeIDE it is needed to use STM32CubeProgrammer to flash the firmware.

Development Toolchains and Compilers

  • STM32CubeIDE, version 1.3.0
  • cmake (version >= 3.13) using arm-gcc compiler delivered with STM32CubeIDE.

Supported Devices and Boards

  • STM32WB55.Nucleo board

Dependencies

  • The cryptographic libraries V3.0.0 used in STM32_WPAN V1.5.0 middleware are not delivered in this package. They must be installed separately in vendors/st/STM32_WPAN/ble/cryptographic/Lib. Use STM32CubeWB version V1.7.0 or earlier to have correct version of cryptographic library.

  • STM32CubeProgrammer version 2.4.0 is required as a post-link processor.

  • This release is pre-integrated with slight adaptations and updates, with the following STM32Cube expansion packages:
